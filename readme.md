## Install

```
$ npm install --global alfred-femine
```

_Requires [Node.js](https://nodejs.org) 8+ and the Alfred [Powerpack](https://www.alfredapp.com/powerpack/)._

## Usage

In Alfred, type `fm`, <kbd>Enter</kbd>, and your query.
